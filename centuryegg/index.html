About
◇ 关于此模板

这套通用界面模板使用纯 CSS Grid 实现两列布局。每个 .block 是一个独立的内容卡片，通过 block-full 跨两列合并，通过内部内容模式类切换显示形态。

CSS 变量体系（:root）将所有颜色、字体、间距参数集中管理，方便二次定制。支持暖色杂志风（#F7F3EC #C4956A）或任意主题色。

设计理念

这套模板基于"内容优先"的设计原则构建。所有视觉元素服务于内容表达，而非喧宾夺主。暖色基底（#F7F3EC）配合衬线/无衬线字体组合（Source Serif 4 + Inter），营造杂志般的阅读体验。

两列网格可根据内容需求自由组合：重要内容用 block-full 跨列突出，次要内容以 block-half 并排展示。移动端自动坍缩为单列流式布局。

每个 .block 内置三种内容模式自由切换，无需改动卡片容器本身。
